# solovision
 A YOLO-based lightweight object tracking system using ByteTrack.


 # How to Install

 Clone the Repo
1. Create a new Python Environment
2. Open this project directory (cd solovision)
3. pip install poetry
4. poetry install


# CLI
python tracking/track.py --yolo-model yolov9m.pt --classes 0 --source (put the video path here) --save --show
